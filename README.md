#### - AUTO WRAP STT - WSTT DAILY

```bash
git clone https://github.com/awkenin/auto-wrap-unwrap-somnia-daily.git && cd auto-wrap-unwrap-somnia-daily
```

### 2. Install
```
npm install
```

### 3. Input private key
```
nano .env
```

### 4. run

```bash
node main.js
```
